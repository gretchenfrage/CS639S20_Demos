Phoenix Kahlo's CLI for sifakis/CS639S20_Demos

USAGE:
    cs39 [MAJOR DEMO NUMBER] [MINOR DEMO NUMBER]
        
        compile and run a demo

    cs39 --list
    
        list available demos
        
    cs39 --help
    
        print this page
        
EXAMPLE:
    cs39 0 2
    
        run LaplacianStencil_0_1
